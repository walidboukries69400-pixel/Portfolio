Band-Pass Filter Design (300 Hz – 3 kHz)

Project Overview

Design and validation of a 5th-order Chebyshev Type I band-pass filter with 0.5 dB ripple. The filter was designed using the insertion-loss method and implemented with R0 = 50Ω source and load impedance.

Design Specifications

  • Lower cutoff frequency: 300 Hz
  • Upper cutoff frequency: 3 kHz
  • Ripple: 0.5 dB
  • Filter type: Chebyshev Type I
  • Order: 5

Calculated Component Values

  • L1 = 5.03 mH
  • C1 = 5.59 µF
  • L2 = 19.5 mH
  • C2 = 1.45 µF
  • L3 = 7.5 mH
  • C3 = 3.75 µF
  • L4 = 19.5 mH
  • C4 = 1.45 µF
  • L5 = 5.03 mH
  • C5 = 5.59 µF

Simulation Results

AC analysis performed in Altium Designer confirmed the expected passband between 300 Hz and 3 kHz. The measured ripple in the passband was consistent with the theoretical 0.5 dB specification.

Practical Validation

The filter was implemented on a breadboard and measured using an oscilloscope. The gain was calculated using:

Gain (dB) = 20 × log(Vout / Vin)

A -6 dB difference between simulation and practical results was observed due to the internal 50Ω source resistance measurement reference.

Conclusion

The band-pass filter behavior matched theoretical predictions. Despite component tolerances and parasitic resistances, the filter maintained correct cutoff frequencies and expected Chebyshev ripple.
